Original monospace browser buttons

Browser buttons are little JavaScriptlets you can drag to your browser's Favorites bar, to add some instant functionality. A famous example of browser buttons are [Google's search buttons].

monospace is proud to introduce to you its very own "monospace browser buttons". While some of these are particularly useful for serious Web developers, others will be of interest to the casual "surfer" as well.


How to use:

Simply drag any of the below yellow hyperlinks to your browser's Favorites bar. The buttons will be available to you no matter what Web site you visit. (Please note that some buttons will only work with Internet Explorer.)


JavaScript Query button:

JavaScript Query

The mother of all monospace browser buttons. Enter any valid JavaScript statement and have it executed. Useful for debugging your own handiwork, or for checking functionality, variables, etc. on other Web pages.


DOM Query button:

DOM Query

This button will present you with a list of all Document Object Model (DOM) elements available on a given Web page. An interesting way to look at the structure of a page. Note that this can be quite slow on large pages. (Internet Explorer only.)


View Cookie button:

View Cookies

Protect your privacy! Use this button to view if and what certain Web sites are writing to your browser's "cookie" file. (A cookie is a piece of persistant information stored on your computer, for use of personalizing a Web site, or worse, for tracking your browsing behavior.)


Set Alarm button:

Set Alarm

Use this button to set an alarm for any Web page. The alarm will be enabled as long as you do not leave the page. Great for auctions!
