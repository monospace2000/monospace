This is an example of blinking text in Internet Explorer. Don't you just LOVE it?

Ok, here's how it works. This piece of script goes in the document head (notice the three variables that you can modify to suit your needs): 

<script type="text/javascript">
<!--
// ------ modify these settings:
blinkForeground = "black";
blinkBackground = "white";
blinkSpeed = 500; // in milliseconds
// ------ end of settings 

blinkFlag = false;
function blink(){
	blinkFlag = !blinkFlag;
	for (var i=0; i<document.all.length; i++){
		if (document.all[i].id.indexOf("blink") != -1){
			if (blinkFlag) document.all[i].style.color = blinkForeground;
			else document.all[i].style.color = blinkBackground;
		}
	}
	window.setTimeout("blink()", blinkSpeed);
}
//-->
</script>

You then enclose the text you want to blink inside a <span> tag, making sure each <span> has a unique id that contains the word "blink", for example blink1, blink2, etc:


This is an example of <span id=blink1>blinking</span> text in Internet Explorer. Don't you just <span id=blink2>LOVE</span> it?

Finally, add a call to the blink() function in the document body's onLoad event handler: 

<body onLoad="blink()">

That's it. Enjoy.

PS: Needless to say, this won't work in Netscape... I'll just leave it to you to modify this script so that Netscape doesn't choke on it :)
